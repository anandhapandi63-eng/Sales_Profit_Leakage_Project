# 📊 Sales Profit Leakage Analysis Dashboard

## Overview

This project is an end-to-end data analytics solution built to identify areas where strong sales performance does not translate into healthy profits.

Using retail transaction data from the Superstore dataset, I analyzed revenue, profit, discounts, product categories, and regional trends to uncover hidden profit leakage and support smarter business decisions.

The final output includes cleaned data, business insights, professional visualizations, and an interactive dashboard.

---

## Business Objective

Many businesses achieve high sales but still experience low profitability due to operational inefficiencies.

This project focuses on answering key business questions:

- Which products generate losses?
- How do discounts affect profit margins?
- Which regions perform best and worst?
- What sales trends are visible over time?
- Where should management take corrective action?
